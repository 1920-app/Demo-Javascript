Éléments de base

Déclaration de variables

En Javascript, la convention de nommage des variables indique qu'il faut utiliser des underscores entre chaque mot du nom (snake case).

      
// Une variable déclarée avec 'var' a une portée de fonction
var une_variable = "Bonjour";

// Une variable déclarée avec 'let' a une portée de bloc
let une_autre_variable = 1;

// Une variable déclarée avec 'const' a une portée de bloc
// Pour la rendre globale, il faut la déclarer en dehors de toute fonction
const une_derniere_variable = true;

La différence entre var et let sera illustrée plus tard, lorsque nous introduirons le concept de fonction.

Opérations algébriques et booléennes

On retrouve les opérations algébriques et booléennes que l'on a l'habitude de rencontrer en Java ou en PHP.

En particulier, on retrouve aussi == et ===, avec la même signification qu'en PHP.

      
var une_variable = 1;
var une_autre_variable = "1";
une_variable == une_autre_variable;  // Ce test est vrai
une_variable === une_autre_variable; // Ce test est faux
