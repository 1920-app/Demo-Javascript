Structures de contrôle

Structures conditionnelles

Les conditions avec if

Rien de nouveau avec les structures conditionnelles utilisant le mot-clé if. L'exemple suivant montre un script écrit en Javascript utilisant une telle structure conditionnelle.

      
let a = 2;
let b = 3;
if (a < b) {
  // La concaténation de chaînes de caractères et de variables se fait avec le symbole +
  // La fonction 'alert' va afficher une fenêtre pop-up avec le message
  alert("Le minimum de " + a + " et " + b + " est " + a);
} else if (a > b) {
  alert("Le minimum de " + a + " et " + b + " est " + b);
} else {
  alert("Les deux valeurs sont égales");
}

Les conditions avec switch

L'instruction switch utilise un test d'égalité strict (===). Son fonctionnement est similaire à ce que l'on a pu voir en Java.

      
switch(my_var) {
  case 1:
    // On fait quelque chose
    break;
  case 2:
    // On fait autre chose
    break;
  // case n:
  default:
    // On fait quelque chose si aucun des tests précédents n'est passé
    break;
}

Structures itératives

Les boucles avec for

En Javascript, on retrouve la distinction entre une boucle for et une boucle for each. Cependant, la boucle for each a un comportement très particulier, différent de ce que l'on a connu jusqu'à présent. En effet, la boucle for each est utilisée pour parcourir des "propriétés énumérables mais qui ne sont pas des symboles" (voir la documentation proposée par Mozilla.) On va donc avoir des exemples d'tuilisation de boucles for each lorsque l'on va introduire les tableaux associatifs. 

      
// Boucle for 'simple'
// Le mot-clé 'let' dans la boucle est une bonne pratique :
// On définit clairement la portée de la variable de boucle que l'on crée
let message = "";
for (let i = 0; i < 10; i++) {
  message += i + "\n";
}
alert(message);

// Boucle foreach qui a un fonctionnement TRÈS spécifique
for (let element in tableau) {
  // Faire quelque chose
}

Les boucles avec while

Les boucles while et do while sont identiques à celles que l'on connaît déjà en Java et PHP. La syntaxe est aussi identique.

      
while (condition) {
  // Faire quelque chose
}

do {
  // Fair quelque chose
} while (condition);
